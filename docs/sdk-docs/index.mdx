## Common Features

All SDKs share these common features:

### Authentication

```javascript
// API Key authentication
const aivo = new Aivo({ apiKey: 'your_api_key' });

// OAuth authentication
const aivo = new Aivo({
  clientId: 'your_client_id',
  clientSecret: 'your_client_secret',
});
```

### Pagination

```javascript
// Manual pagination
const page1 = await aivo.lessons.list({ limit: 20 });
const page2 = await aivo.lessons.list({ limit: 20, cursor: page1.pagination.cursor });

// Auto-pagination (iterates through all pages)
for await (const lesson of aivo.lessons.listAutoPaging()) {
  console.log(lesson.title);
}
```

### Error Handling

```javascript
import { AivoError, NotFoundError, RateLimitError } from '@aivo/sdk';

try {
  const lesson = await aivo.lessons.get('les_invalid');
} catch (error) {
  if (error instanceof NotFoundError) {
    console.log('Lesson not found');
  } else if (error instanceof RateLimitError) {
    console.log(`Retry after ${error.retryAfter} seconds`);
  }
}
```

### Webhooks

```javascript
import { verifyWebhookSignature } from '@aivo/sdk';

// Verify webhook signature
const isValid = verifyWebhookSignature(requestBody, signatureHeader, webhookSecret);
```
