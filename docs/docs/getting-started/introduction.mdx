# Welcome to AIVO

The AIVO Platform API enables you to build powerful educational experiences for learners of all ages. Whether you're building a learning management system, a tutoring app, or an educational game, our API provides everything you need.

## What is AIVO?

AIVO is a comprehensive educational platform that provides:

- **Learning Content Management** - Create, organize, and deliver interactive lessons and courses
- **Adaptive Learning** - AI-powered personalized learning paths based on student performance
- **Progress Tracking** - Real-time monitoring of student progress and skill mastery
- **Gamification** - XP, achievements, leaderboards, and streaks to keep learners engaged
- **Analytics** - Comprehensive insights into learning outcomes and engagement
- **Compliance** - Built-in COPPA, FERPA, and GDPR compliance

## API Environments

| Environment    | Base URL                          | Purpose                 |
| -------------- | --------------------------------- | ----------------------- |
| **Production** | `https://api.aivo.edu/v2`         | Live production data    |
| **Sandbox**    | `https://sandbox-api.aivo.edu/v2` | Testing and development |

:::tip Always Use Sandbox First
Develop and test your integration in the sandbox environment before going live.
Sandbox data is reset weekly and doesn't affect real users.
:::

## Authentication

The API supports two authentication methods:

<Tabs>
<TabItem value="bearer" label="Bearer Token" default>

```bash
curl -X GET "https://api.aivo.edu/v2/lessons" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

</TabItem>
<TabItem value="apikey" label="API Key">

```bash
curl -X GET "https://api.aivo.edu/v2/lessons" \
  -H "X-API-Key: YOUR_API_KEY"
```

</TabItem>
</Tabs>
